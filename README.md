# skillforge-ai
AI Based Skill Recommendation System For Job Seekers

## Backend options

- Vercel serverless API routes live under `api/` (Node.js).
- Optional Python Flask backend (Postgres-backed) lives under `backend/` (see `backend/README.md`).
